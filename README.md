## About <a name="about"></a>

HBO Clone web app consuming tmdb api.

### Requirements

- NodeJS
- NextJS 13
- Internet (for documentation)
- VSCode

### Installation

Clone this repository

```
git clone https://github.com/andeilsongf/hbo-clone.git
```

Get inside the folder project

```
cd hbo-clone
```

Run Project

```
npm run dev
```
